# ChengDa

### Introduction
客製化進銷存管理系統

### Development Tools 
- Visual Studio 2017
- MSSQL 2010

### Function Description

- 會員專區
  + 登出入作業
  * 客製化統計顯示
  * 查詢/新增/修改/刪除作業
    
- 行銷專區
  * 查詢/新增/修改/刪除作業
  * 客製化統計顯示
  
- 進銷存專區
  * 進貨/出貨/寄庫/退貨
  * 查詢/新增/修改/刪除作業
  * 客製化統計顯示
  
- 交易參數設定
  * 片語類別設定
  * 查詢/新增/修改/刪除作業
  
- 資料庫備份
  * 匯出 .bak 備份檔
